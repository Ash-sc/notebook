# notebook
notebook with Vue &amp; Node &amp; MongoDB

# Environment
**Node.js**、**MongoDB**

# 如何运行

``` bash
# 安装依赖
npm install

# 启动mongoDB，如果没有mongoDB请先安装 (可在server/mongo/index.js中修改mongo连接设置)
# 服务端启动
node ./server/node-app.js

# 前端服务启动
npm start
# 可在build/dev.js 和 build/config 文件夹中修改代理设置 和webpack相关设置
```
